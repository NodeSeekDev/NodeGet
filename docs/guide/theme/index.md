# 公开探针页面

NodeGet 的公开展示页面是一个纯静态网页服务，可以部署到任意的静态文件托管服务

开发者可以使用任意自己擅长的技术框架来编写 NodeGet 主题，并通过 [主题分发服务](/dev/theme/distribution.md) 一键分享给需要的用户

最开始的时候，NodeGet 不想限制公开主题页面的开发要求，只提供了一个简易的官方主题，并说明了其工作原理

由于这份说明更像是给开发者看的开发说明，对于普通用户部署 NodeGet 主题存在一定的门槛，因此官方改进了部署方案，降低了部署门槛

对于符合 [NodeGet主题框架](/dev/theme/) 的主题，仅推荐两种部署方式：

- 对于不追求折腾的用户，建议通过[一键部署](./quick-install.md)来部署主题
- 对于有一定的折腾能力的用户，建议通过 [cloudflare pages](./cloudflare.md)来部署主题
